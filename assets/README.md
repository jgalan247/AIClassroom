# Assets Folder

This folder contains static assets for the AI Classroom website.

## Required Files

Please place the following files in this directory:

1. **logo.png** - Coderra logo for footer display
   - Recommended size: Height of ~80-100px for optimal display (will be scaled to 40px)
   - Format: PNG with transparent background preferred

2. **favicon.ico** - Website favicon for browser tabs
   - Size: 16x16 or 32x32 pixels
   - Format: ICO file
